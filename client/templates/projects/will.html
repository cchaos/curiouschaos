WILL Styles

I started working at WILL in 2007, straight out of college. I was immediately tasked with creating a new logo for the company. Then in 2015, we once again decided to update the branding so it more cohesively exuded who WILL is. Just before leaving will in early 2016, we finalized the new logo. The broken arrow represents the multiple pathways through WILL's interactive storylines.

Style Guide & Documentation

To maintain this new brand throughout our custom applications, as a development team, we kept a living style guide with detailed documentation.

Component-Based Development

We built the more complex components as Ruby components and helpers for consistency and maintainability. The documentation would then show the resulting component, the code for ruby helper, and the resulting HTML.

To help with future iterations and implementation of the style guide, we maintained the components in a Ruby gem that could be used by all of our applications. This way, when we altered the HTML and CSS in one place they would update across all applications.

WILL Apps

The WILL Apps suite is a compilation of custom applications that help the whole company in day-to-day operations. They give all employees access to a customized issue tracker, client review and user testing platform, and public or private program access control.
